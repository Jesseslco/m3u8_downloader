# m3u8_downloader
asynchronous download video via m3u8 url or m3u8 file

 
## Requirments

#### Package
* python3.8 required
* ffmpeg required

#### Modules
* requests
* m3u8
* asyncio
* aiohttp
* aiofile
* ffmpeg-python
> you can install via `pip install -r requirements.txt`
## How to use
1. clone the project
2. cd M3U8/
3. python3.8 manage.py
